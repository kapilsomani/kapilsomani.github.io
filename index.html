MultiClient Chatroom Application

Implemented a chat room among multiple clients. Functionality is as described below:
- When the client requests the connection, the server prompts for the client’s nickname. Thus, each client should register with the server with users nickname.
- The messages on the chat room shows the nickname of the person posting the message.
- The messages posted in chatroom are broadcasted to all the clients in realtime.
- Redis is used to provide persistent storage of messages, which allows later users to view initial messages posted in the chatroom.

CAPTCHA

- A Java application which extracts text from CAPTCHA images. Vulnerabilities in CAPTCHA images (which are still used by many websites) were highlighted by breaking some CAPTCHAs.
